# Arabic Flashcard App

A Python Tkinter app to learn Arabic vocabulary.

## Features
- Flashcards (Arabic → English)
- Auto flip after 3 seconds
- Progress tracking
- Save learned words

## Run
python3 main.py






## How to Run

```bash
git clone https://github.com/kabirbapson/arabic-flashcard.git
cd arabic-flashcard

python3 -m venv venv
source venv/bin/activate

pip install pandas arabic-reshaper python-bidi

python3 main.py
```
